#  TODO-LIST Application

My first Full-Stack Development project built after completing a Full-Stack Development course.

##  Features

- User Signup & Login Authentication
- Password Encryption using bcrypt
- Add, Edit and Delete Tasks
- Mark Tasks as Completed
- Undo Completed Tasks
- Search Tasks
- Filter Tasks (All, Completed, Pending)
- Due Today Widget
- Progress Tracking
- Task Deadlines
- Email Reminder Notifications
  - 24 Hours Before Deadline
  - 3 Hours Before Deadline
  - 15 Minutes Before Deadline

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Additional Tools
- Nodemailer
- Node Cron
- bcryptjs
